Home Styler

Live Preview Documentation

Functional Requirements

  1. Home Page: Displays a design gallery and menus for different features, using either a table or image gallery.
  2. Viewing Design Styles: Users can view design styles by categories (e.g., Living Room, Kitchen). Each design page includes images, descriptions, and product lists.
  3. Professional Interior Designers Gallery: Users can view designers' details for contact.
  4. Searching, Sorting, and Filtering: Users can search, sort, and filter designs by various criteria.
  5. Feedback: A form allows users to provide feedback (data not saved).
  6. Contact Us: Displays contact info, with optional GPS/geolocation.
  7. Browsing Products: Allows browsing interior products by type, including categories like Furniture, Lighting, Decor, Kitchen, and Bathroom.
  8. Product Selection: Users can filter products, add selections to a wishlist.
  9. User Reviews and Ratings: Users can view reviews and ratings for designs and products.
  10. About Us Page: Provides information about the website creators.
  11. Sitemap: A sitemap is provided on the home page to guide users through the site.
